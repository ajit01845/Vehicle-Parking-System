# 🚗 Vehicle Parking Management System

A full-stack Vehicle Parking Management System developed using **Python, Django, SQLite, HTML, CSS, Bootstrap, and JavaScript**. The system provides separate dashboards for **Admin**, **Manager**, and **User** with secure authentication and role-based access.

---

# 📖 Project Overview

The Vehicle Parking Management System is designed to simplify parking operations by allowing users to reserve parking spaces, managers to monitor daily parking activities, and administrators to manage the complete parking system.

The application supports multiple user roles, parking management, payment tracking, notifications, security monitoring, and reporting.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/ajit01845/Vehicle-Parking-System.git
```

## Move into Project

```bash
cd Vehicle-Parking-System
```

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Apply Migrations

```bash
python manage.py migrate
```

## Run Server

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
